# Description
제작 기간: 2018.07 ~ 2018.08 <br>
<br>
구성: 카메라, 라즈베리파이, 노트북, 로봇팔, 레일, 스텝 모터<br>
<br>
천장과 정면에 부착된 카메라를 통해 탁구공의 움직임을 촬영하고, 영상 처리를 통해 탁구공의 예상 경로를 계산했다. 탁구공이 라켓으로 칠 수 있는 범위 안에 들어왔다고 판단되면, 라켓을 휘두르는 방식으로 동작한다.
